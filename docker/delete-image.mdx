## Introduction
In `docker`, you can quickly `delete` local `images` through rmi, which can be deleted by image id or image name

## Command Line
delete `mysql` image
```bash
docker rmi mysql
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| rmi   |Remove one or more images.|
