## Introduction
There are a large number of high-quality `images` in the `docker` hub, which can be downloaded and used directly.
## Command Line
`Pull` the `mysql` image
```bash
docker pull mysql
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| pull   |Pull an image or a repository from a registry.|
