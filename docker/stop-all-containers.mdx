## Introduction
In docker, you can quickly stop all containers with the `stop` command
## Command Line
stop all containers
```bash
docker stop $(docker ps -q)
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| stop   |Stop one or more running containers.|
| ps   |List containers.|
| -q   |Only display container IDs.|
