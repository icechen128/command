## Introduction
In centos8, `jdk` can be quickly installed through `yum`
## Command Line
Install `jdk` using the `yum` tool
```linux
yum install -y java-1.8.0-openjdk
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| yum   |a package manager.|
| install   |install a package or packages on your system.|
