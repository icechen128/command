## Introduction
In linux, you can connect to other servers through the `ssh` command
## Command Line
Connect to the `192.168.91.135` server and log in with the `root` account
```linux
ssh root@192.168.91.135
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| ssh   |provides a secure encrypted connection between two hosts over an insecure network.|
