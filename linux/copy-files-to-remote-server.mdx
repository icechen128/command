## Introduction
In Linux, you can copy files to a remote server through the `scp` command
## Command Line
Copy the `kafka_2.12-3.2.1.tgz` file to the `/home/cmdhelp/test` directory of the remote server `192.168.91.135`
```linux
scp kafka_2.12-3.2.1.tgz root@192.168.91.135:/home/cmdhelp/test
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| scp   |Commands to copy files remotely under Linux.|
