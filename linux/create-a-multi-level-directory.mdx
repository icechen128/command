## Introduction
In Linux, you can quickly create multi-level directories through `mkdir`
## Command Line
Create a multilevel directory `zk/data`
```linux
mkdir -p zk/data
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| mkdir   |Create the DIRECTORY(ies), if they do not already exist.|
| -p   |no error if existing, make parent directories as needed.|
