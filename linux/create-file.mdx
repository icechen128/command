## Introduction
In linux, you can quickly create a file by `touch`
## Command Line
Create a file called `text.txt`
```linux
touch text.txt
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| touch   |Update the access and modification times of each FILE to the current time.|
