## Introduction
In linux, you can quickly create a new user with `adduser`

## Command Line
Create a user named `cmdhelp`
```bash
sudo adduser cmdhelp
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| sudo   |execute a command as another user|
| adduser   |Add user.|
