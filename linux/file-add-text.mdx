## Introduction
In Linux systems, you can quickly write text to a file through `echo`
## Command Line
Add `add test` to the end of the file `target.txt`
```bash
echo "add test" >> target.txt
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| echo   |Print the value of the shell variable in the shell, or directly output the specified string.|
