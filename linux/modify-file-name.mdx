## Introduction
In linux system, you can rename a file or folder by `mv`
## Command Line
Rename the `text.txt` file to `target.txt`
```bash
mv text.txt target.txt
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| ls   |Rename SOURCE to DEST, or move SOURCE(s) to DIRECTORY.|
