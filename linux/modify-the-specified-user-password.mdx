## Introduction
In linux, you can quickly change user passwords through `passwd`
## Command Line
Change the password of the user `cmdhelp`
```bash
sudo passwd cmdhelp
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| sudo   |execute a command as another user|
| passwd |set password.|

###  operation result
```
[sudo] password for root: 
Changing password for user cmdhelp.
New password: 
BAD PASSWORD: The password is shorter than 8 characters
Retype new password: 
passwd: all authentication tokens updated successfully.
```
### Notice
If the password is less than 8 digits, you can continue to enter it for the second time.
