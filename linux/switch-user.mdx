## Introduction
In linux system, you can quickly switch users through `su`
## Command Line
Switch user to `cmdhelp`
```linux
su cmdhelp
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| su   |Change the effective user ID and group ID to that of <user>.|
