## Introduction
In linux system, you can check the current time through the `date` command
## Command Line
View current system date
```bash
date
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| date   |Display the current time in the given FORMAT, or set the system date.|
###  operation result
```
Tue Aug 23 14:28:36 CST 2022
```
