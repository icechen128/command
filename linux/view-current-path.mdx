## Introduction
In linux, you can use the `pwd` command to view the current path.
## Command Line
View the current path
```linux
pwd
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| pwd   |    Print the name of the current working directory.|

###  operation result
```
/home/cmdhelp/test
```
