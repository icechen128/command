## Introduction
In linux, you can use the `crontab` command to view all the scheduled tasks of the current user
## Command Line
View automated tasks for the current user
```linux
crontab -l
```
## Parameter
| parameter | Introduction                                                               |
| --------- | -------------------------------------------------------------------------- |
| crontab   |scheduled tasks.|
| -l   |list user's crontab.|
###  operation result
```
no crontab for root
```
